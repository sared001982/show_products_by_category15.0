SHOW PRODUCTS GROUPED BY CATEGORY ON SALE ORDER REPORT

This Odoo module enhances the Sale Order printout by grouping order lines based on the product categories. For each group, a clear header with the category name is displayed, improving readability and providing better organization in your printed documents.

Key Features

  • Groups sale order lines by product category
  • Displays the category name as a section header
  • Compatible with Odoo 15
  • No configuration required â€“ works out of the box
  • Improves clarity in sale order documents for customers and internal teams

Use Case

Imagine a business that sells products in multiple categories (e.g., Electronics, Furniture, Office Supplies). With this module, customers receive sale orders that are easier to understand, with items grouped under their respective categories.

Installation

  1. Download or clone the module into your Odoo 15 custom addons path
  2. Update the app list
  3. Install the module from the Odoo interface
