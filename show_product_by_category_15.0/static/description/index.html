🧾 Print Sale Orders Grouped by Product Category

This Odoo module enhances the default sale order report by grouping sale order lines based on their Product Category. Each category is printed as a separate section with its name as a header, making the report more structured and readable, especially for orders with many products.

🌟 Features

  • Group sale order lines by product category
  • Display category names as section headers in the report
  • Clean and professional PDF layout
  • No changes to existing workflows—just install and print

🧩 Compatibility

Tested on Odoo 15.0 Community and Enterprise editions.

🖼️ Screenshots

📦 Installation

Simply install the module like any other. No configuration needed.

🧑‍💻 Usage

After installation, go to any Sale Order and click on Print → Quotation / Order. The report will be generated with lines grouped by category.
